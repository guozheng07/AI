⚠️ 技术难点
① 三路轮询协调
生成轮询 + 诊断轮询 + 进行中任务轮询同时运行。使用非响应式 _diagnosisAborted、_cancelled flag 控制中止，避免触发额外视图更新。
② fetchPicDetail 竞态保护
快速切换历史记录时多请求乱序返回，用 _fetchPicDetailSeq 序号递增，每次只采用最新请求结果，过期结果直接丢弃。
③ 参数标签二行截断
_measureParamTags()：标签宽度不均等，无法字数截断。通过插入临时 probe 节点测量 offsetTop，二分查找让 +N 落在第二行末尾。
④ 进行中任务占位
后端未返回结果前，前端维护 runningPicList，在 mergedHistoryList computed 中置顶，展示原图 + loading 蒙层。

⚠️ 技术难点
① 多图历史快照
历史接口只存每个任务的代表图（非全部），生成完成时立即保存 initialPicList = [{picUrl, picId}] 完整快照供历史面板展示。
② Promise.allSettled 并行聚合
历史面板同时请求历史列表和进行中任务，allSettled 保证一个失败另一个仍展示，不整体报错。
③ 场景分组 + 缩略图映射
sceneGroups 分「基础/更多」两组，通过缩略图 URL 映射渲染预览图，场景切换高亮选中。
④ 全局点击关闭自定义下拉
mounted 注册 document.click，beforeDestroy 移除，统一关闭数量/模型两个下拉面板。
🌐 语言专家（LanguageSpec）LanguageSpec.vue · 约 982 行
多语言机器翻译工具，支持中 / 英 / 阿三语互译。核心特性：用户选中页面任意文本后，翻译结果自动填充并触发，无需手动复制粘贴。
功能点
  • 自动检测源语言（中/英/阿，字符集统计）
  • 支持同时翻译多个目标语言（默认英文 + 阿拉伯文）
  • 翻译结果卡片叠加展示，每次追加不覆盖历史
  • 结果卡片内支持切换显示的目标语言
  • 一键复制（Clipboard API + execCommand 降级）
  • RTL 支持：阿拉伯文卡片自动 direction: rtl
  • 支持互换源/目标语言（单目标时有效）
  • 中途停止翻译（abortFlag 控制）
语言自动检测算法
arCount = text.match(/[\u0600-\u06FF]/g)?.length
enCount = text.match(/[a-zA-Z]/g)?.length
zhCount = text.match(/[\u4E00-\u9FFF]/g)?.length
// 字符数最多的 → 检测为源语言
⚠️ 技术难点
① Vue 2 响应式陷阱
切换卡片内语言时直接修改数组成员不会触发更新，必须用 this.$set(resultCards, idx, newCard) 替换整个对象。
② RTL 文本渲染
阿拉伯语从右到左，结果卡片根据语言类型动态切换 direction: rtl; text-align: right。
③ 剪贴板 API 降级
iframe 内嵌环境 navigator.clipboard 可能被限制，失败时降级创建隐藏 textarea + execCommand('copy')。
④ 全量翻译快照
每张卡片保存 _fullTranslations（全量）+ translations（当前显示）。切换语言时从全量筛选，无需重新请求接口。

waitForRefAndFill 异步等待机制
异步组件首次渲染时 $nextTick 不足以等待 $refs 挂载，需使用轮询重试：每 100ms 检查一次，最多 30 次（共 3 秒）。 
waitForRefAndFill(type, data, attempt = 0) {
  const ref = this.$refs[refName];
  if (ref && ref.fillData) {
    ref.fillData({ ...data, subAppContentWindow });
    return;
  }
  // 最多重试 30 次（每次 100ms，共 3s）
  if (attempt < 30) {
    setTimeout(() => this.waitForRefAndFill(type, data, attempt + 1), 100);
  }
}

10关键设计决策
跨 IFRAME 通信
window.postMessage
外壳与子应用独立 JS 上下文，无法直接访问 DOM/变量
异步组件等待策略
轮询重试（100ms × 30次）
$nextTick 不足以等待异步组件首次渲染完成，需主动轮询 $refs
模块显隐方式
v-show + $refs（非 v-if）
v-if 会销毁组件，切换时 $refs 为 null；v-show 保持组件挂载
图片回填 iframe 定位
优先 event.source，降级 DOM 查找
页面可能有多个 iframe，缓存唤醒来源 contentWindow 最精准，DOM 查找作为兜底
翻译工具唤醒方式
全局 AITextSelector 插件
无需修改各业务页面，低侵入；通过 Vue.use() 一次性注册全局生效
AI 菜单 DOM 挂载位置
根 <div>（xx-layout 外层）
xx-layout 内层有 overflow:hidden，若菜单挂在其内部会被裁剪不可见
AI 浮层 z-index 管理
独立变量 $z-header-ai-overlay
低于 Element UI 默认 2000+，不遮盖语言/地区下拉；通过 SCSS 变量统一管理
